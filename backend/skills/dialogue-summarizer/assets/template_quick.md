# 快速总结模板

## 适用场景
- 简短对话（≤5轮）
- 日常交流回顾
- 快速信息提取
- 即时反馈需求

## 模板结构

```
## 快速对话总结

### 📋 基本信息
- **对话时间**: {{timestamp}}
- **对话轮数**: {{turn_count}}
- **主要话题**: {{main_topic}}

### 🎯 关键要点
{{#each key_points}}
{{@index}}. {{this}}
{{/each}}

### 📝 行动项
{{#if has_actions}}
{{#each actions}}
- {{this.action}} ({{this.assignee}})
{{/each}}
{{else}}
- 无
{{/if}}

### 💡 简要建议
{{suggestion}}
```

## 字段说明

### 基本信息
- **timestamp**: 对话时间，格式：YYYY-MM-DD HH:MM
- **turn_count**: 对话总轮数
- **main_topic**: 主要话题，1-3个关键词

### 关键要点
- 提取最重要的3-5个信息点
- 按重要性排序
- 每点不超过20字

### 行动项
- 识别出的待办事项
- 包含负责人信息
- 最多显示3个最重要的行动项

### 简要建议
- 基于对话内容的实用建议
- 1-2句话，简洁明了
- 可操作的具体建议

## 使用示例

### 示例对话
```
用户: 今天天气怎么样？
AI: 北京今天15°C，晴朗，适合外出
用户: 谢谢，那我下午去公园
```

### 填充后的总结
```
## 快速对话总结

### 📋 基本信息
- **对话时间**: 2024-03-18 14:30
- **对话轮数**: 3
- **主要话题**: 天气查询

### 🎯 关键要点
1. 用户查询北京天气
2. AI提供天气信息：15°C，晴朗
3. 用户计划下午去公园

### 📝 行动项
- 无

### 💡 简要建议
天气良好，适合户外活动，注意防晒。
```

## 生成规则

### 1. 话题提取规则
- 从对话中提取出现频率最高的名词
- 优先提取专有名词和具体概念
- 最多选择3个话题

### 2. 关键点选择规则
1. **问题-回答对**: 用户提问和AI回答
2. **决策点**: 明确的决定或选择
3. **行动意向**: 用户表达的行动计划
4. **重要信息**: 数字、日期、关键事实

### 3. 行动项识别规则
- 仅识别明确的行动意向
- 包含时间或责任信息的优先
- 过滤日常问候和礼貌用语

### 4. 建议生成规则
- 基于对话内容提供实用建议
- 避免重复对话中已有的信息
- 保持积极和建设性

## 优化提示

### 语言风格
- 使用简洁明了的语言
- 避免专业术语
- 使用积极语气

### 格式优化
- 使用适当的标题层级
- 合理使用列表和项目符号
- 保持一致的格式风格

### 内容质量
- 确保信息准确无误
- 突出重点，避免冗余
- 提供有价值的见解
