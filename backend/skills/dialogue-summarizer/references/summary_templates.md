# 总结模板参考文档

## 📋 模板概述

本技能提供三种总结模板，适用于不同场景和对话长度。

## 🚀 快速总结模板

**适用场景**: 简短对话（≤5轮）、快速回顾、日常交流

### 模板结构
```
## 快速对话总结

### 📋 基本信息
- **对话时间**: [时间]
- **对话轮数**: [数字]
- **主要话题**: [话题]

### 🎯 关键要点
1. [要点1]
2. [要点2]
3. [要点3]

### 📝 行动项
- [行动项1]
- [行动项2]

### 💡 简要建议
[建议内容]
```

### 使用示例
**对话**: 
- 用户: "今天天气怎么样？"
- AI: "北京今天15°C，晴朗"
- 用户: "谢谢"

**总结**:
```
## 快速对话总结

### 📋 基本信息
- **对话时间**: 2024-03-18 14:30
- **对话轮数**: 3
- **主要话题**: 天气查询

### 🎯 关键要点
1. 用户查询北京天气
2. AI提供天气信息：15°C，晴朗
3. 用户表示感谢

### 📝 行动项
- 无

### 💡 简要建议
天气信息查询完成，无需进一步行动。
```

   - 可以更早询问具体的输出格式偏好
   - 可以提供更多定制化选项

3. **下次对话建议**:
   - 先测试技能的实际效果
   - 收集用户对总结格式的反馈

## 🎨 模板选择指南

### 选择标准
1. **对话长度**:
   - ≤5轮: 快速总结
   - 6-15轮: 标准总结  
   - ≥16轮: 详细复盘

2. **对话复杂度**:
   - 简单日常: 快速总结
   - 技术讨论: 标准总结
   - 战略规划: 详细复盘

3. **用户需求**:
   - 快速回顾: 快速总结
   - 正式记录: 标准总结
   - 深度分析: 详细复盘

### 混合使用建议
对于特别复杂的对话，可以考虑:
1. **分层总结**: 先用快速总结给出概览，再用详细复盘深入分析
2. **渐进披露**: 先提供标准总结，根据用户需求再提供详细复盘
3. **定制组合**: 混合不同模板的部分，创建定制化总结
