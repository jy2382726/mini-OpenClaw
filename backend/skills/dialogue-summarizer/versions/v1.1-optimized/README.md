# Dialogue Summarizer 技能

## 📋 概述

Dialogue Summarizer 是一个智能对话总结技能，能够对对话上下文进行智能总结和复盘，提取关键信息、识别行动项、生成结构化回顾。

## 🎯 功能特点

### 核心功能
- **多模式总结**: 支持快速、标准、详细三种总结模式
- **智能上下文处理**: 自动处理对话上下文，支持回退机制
- **关键信息提取**: 自动提取关键点、行动项、决策信息
- **结构化输出**: 提供标准化的总结报告模板

### 高级功能
- **上下文回退**: 当上下文不完整时，智能请求补充信息
- **话题识别**: 自动识别对话主题和子话题
- **行动项跟踪**: 识别和跟踪待办事项
- **数据统计**: 提供对话统计数据和分析

## 🔧 技术实现

### 核心算法
1. **上下文收集**: 从系统获取对话历史，支持分段处理
2. **话题分析**: 识别对话主题和子话题
3. **信息提取**: 提取关键点、行动项、决策信息
4. **模板填充**: 根据模式选择合适的模板生成总结

### 处理流程
```
收集上下文 → 分析对话结构 → 选择总结模式 → 生成结构化总结 → 优化输出
```
