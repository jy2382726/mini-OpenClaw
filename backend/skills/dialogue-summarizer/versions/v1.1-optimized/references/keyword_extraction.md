# 关键词提取方法参考文档

## 🎯 概述

本文档定义了对话总结中使用的关键词提取规则和方法，用于识别重要信息、行动项、决策点等。

## 🔍 提取规则

### 1. 上下文感知提取
- **位置权重**: 对话开头和结尾的关键词权重更高
- **重复检测**: 重复出现的关键词重要性更高
- **组合模式**: 多个关键词组合出现时提取完整短语

### 2. 模式匹配规则

#### 行动项模式
```
pattern: (需要|应该|将要|计划|安排|负责|完成) + (.+?)(?:。|！|!|；|;|$)
示例: "需要创建一个新的技能" → 提取: "创建一个新的技能"
```

#### 决策模式
```
pattern: (决定|同意|选择|确定|确认|达成) + (.+?)(?:。|！|!|；|;|$)
示例: "决定采用第一种方案" → 提取: "采用第一种方案"
```

#### 问题模式
```
pattern: (问题|困难|挑战|障碍|风险) + 是? + (.+?)(?:。|！|!|；|;|$)
示例: "问题是时间不够" → 提取: "时间不够"
```

### 3. 优先级计算

#### 权重分配
- **关键词类型权重**:
  - 行动关键词: 1.2
  - 决策关键词: 1.1
  - 问题关键词: 1.0
  - 技术关键词: 0.9
  - 项目关键词: 0.8

- **位置权重**:
  - 对话前20%: 1.3
  - 对话中间60%: 1.0
  - 对话后20%: 1.2

- **重复权重**:
  - 首次出现: 1.0
  - 第二次出现: 1.2
  - 第三次及以上: 1.5

#### 优先级公式
```
优先级 = 类型权重 × 位置权重 × 重复权重 × 长度因子
长度因子 = min(1.0, 关键词长度 / 20)
```

## 🛠️ 提取算法

### 1. 基础提取流程
```python
def extract_keywords(text, keyword_categories):
    """
    从文本中提取关键词
    """
    keywords = []
    
    for category, word_list in keyword_categories.items():
        for word in word_list:
            if word in text:
                # 提取上下文
                start = max(0, text.find(word) - 20)
                end = min(len(text), text.find(word) + 50)
                context = text[start:end]
                
                keywords.append({
                    'keyword': word,
                    'category': category,
                    'context': context,
                    'position': text.find(word)
                })
    
    return keywords
```

#### 实体识别
```python
def extract_entities(text):
    """
    识别特定类型的实体
    """
    entities = {
        'dates': [],
        'numbers': [],
        'names': [],
        'urls': [],
        'emails': []
    }
    
    # 日期提取
    date_patterns = [
        r'\d{4}年\d{1,2}月\d{1,2}日',
        r'\d{1,2}月\d{1,2}日',
        r'\d{4}-\d{1,2}-\d{1,2}',
        r'明天|今天|昨天|下周|下月|明年'
    ]
    
    for pattern in date_patterns:
        matches = re.findall(pattern, text)
        entities['dates'].extend(matches)
    
    # 数字提取
    number_patterns = [
        r'\d+',
        r'\d+\.\d+',
        r'\d+%',
        r'第\d+[个条项点]'
    ]
    
    for pattern in number_patterns:
        matches = re.findall(pattern, text)
        entities['numbers'].extend(matches)
    
    return entities
```

## 🔧 优化建议

### 1. 动态词库
- 根据对话领域动态调整关键词权重
- 学习用户常用术语并加入词库
- 支持自定义关键词列表

### 2. 上下文关联
- 建立关键词之间的关联关系
- 识别关键词链（如：问题→解决方案→行动项）
- 分析关键词的时间序列

### 3. 多语言支持
- 支持中英文关键词混合提取
- 处理同义词和近义词
- 支持领域特定术语

### 4. 机器学习增强
- 使用模型识别复杂模式
- 自动发现新的关键词类别
- 优化提取准确率和召回率

## 📈 性能指标

### 提取质量指标
- **准确率**: 提取的关键词中真正相关的比例
- **召回率**: 所有相关关键词中被提取出来的比例
- **F1分数**: 准确率和召回率的调和平均

### 效率指标
- **处理速度**: 每千字处理时间
- **内存使用**: 峰值内存占用
- **可扩展性**: 支持的最大文本长度
