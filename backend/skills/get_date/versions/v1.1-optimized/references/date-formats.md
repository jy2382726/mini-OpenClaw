# Date Command Format Reference

本文档提供 `date` 命令的常用格式字符串参考，供脚本开发和调试使用。

## 基础格式

| 格式符 | 说明 | 示例输出 |
|--------|------|----------|
| `%Y` | 四位年份 | 2024 |
| `%y` | 两位年份 | 24 |
| `%m` | 月份 (01-12) | 03 |
| `%d` | 日期 (01-31) | 17 |
| `%H` | 小时 (00-23) | 14 |
| `%M` | 分钟 (00-59) | 30 |
| `%S` | 秒 (00-59) | 45 |

## 星期相关

| 格式符 | 说明 | 示例输出 |
|--------|------|----------|
| `%A` | 完整星期名 (英文) | Monday |
| `%a` | 缩写星期名 (英文) | Mon |
| `%u` | 星期数字 (1-7, 1=周一) | 1 |
| `%w` | 星期数字 (0-6, 0=周日) | 1 |

## 常用组合

```bash
# ISO 8601 格式
date +%Y-%m-%d          # 2024-03-17
date +%Y-%m-%dT%H:%M:%S # 2024-03-17T14:30:45

# 中文常用格式
date +%Y年%m月%d日      # 2024 年 03 月 17 日
date +%m/%d/%Y          # 03/17/2024

# 时间戳
date +%s                # 1710684645
date +%s%N              # 1710684645123456789
```

## 相对日期计算

```bash
# 昨天
date -d "yesterday" +%Y-%m-%d

# 明天
date -d "tomorrow" +%Y-%m-%d

# 一周后
date -d "+7 days" +%Y-%m-%d

# 一个月前
date -d "-1 month" +%Y-%m-%d
```

## 时区相关

```bash
# 设置时区
TZ=Asia/Shanghai date +%Y-%m-%d\ %H:%M:%S
TZ=UTC date +%Y-%m-%d\ %H:%M:%S

# 显示时区信息
date +%Z                # CST
date +%z                # +0800
```

## 注意事项

1. **GNU date vs BSD date**: 
   - Linux 使用 GNU date，支持 `-d` 参数进行相对日期计算
   - macOS 使用 BSD date，使用 `-v` 参数进行日期计算

2. **本地化**: 
   - 星期和月份名称受 `LANG` 环境变量影响
   - 中文环境下 `%A` 可能输出中文星期名

3. **性能**: 
   - 每次调用 `date` 都会启动新进程
   - 批量获取多个时间值时，建议单次调用获取所有需要的格式
