# History Schema

## Summary Record
- `run_id`
- `skill_targets`
- `models`
- `aggregate_scores`
- `final_verdict`
- `timestamp`

## Trend Record
- `skill_name`
- `run_count`
- `latest_verdict`
- `trend_signal`
- `models_seen`
- `last_updated`

## Consistency Rule
Trend review is only meaningful when benchmark scenarios and scoring rules remain comparable across runs.
